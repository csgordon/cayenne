Getting Cayenne

You can get Cayenne with FTP from ftp://ftp.cs.chalmers.se/users/cs/augustss/cayenne.tgz . To build and install cayenne do 
        gunzip < cayenne.tgz | tar xf -
        cd cayenne
        make
        make install
For Cayenne to work you need hbc and lmlc which can be found at ftp://ftp.cs.chalmers.se/pub/haskell/chalmers/.

There is some documentation in the html/ directory and some sample programs in test/.

Thing are changing daily, so beware!
